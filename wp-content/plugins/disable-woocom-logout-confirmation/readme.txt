== Description ==

This lightweight plugin disables woocommerce logout confirmation!

If you have any queries, issues or feedback regarding the plugin, please contact developer at <a href="http://hemthapa.com?wcdislogout" target="_blank">hemthapa.com</a>

== Installation ==

1. Install the plugin from within the Plugin manager or upload the directory `disable-wc-logout-confirmaton` and all of its contents to the `/wp-content/plugins/` directory of your website

2. Activate the plugin through the 'Plugins' menu in WordPress

3. That's all.
